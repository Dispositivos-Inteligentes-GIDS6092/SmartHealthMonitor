# SmartHealth Monitor
Aplicación Android multiplataforma diseñada para el monitoreo personal de indicadores de salud.Proyecto integrador desarrollado en la Universidad Tecnológica del Norte de Guanajuato (UTNG) — 9° Cuatrimestre, periodo 2025.

Stack tecnológico

    Kotlin + Jetpack Compose
    Material Design 3
    API de Capa de Datos para Dispositivos Portátiles (Wear OS)
    Android TV / Leanback + Media3
    Jetpack Navigation + Room + StateFlow

Pantallas y Nave
📱 Pantallas y funcionalidades implementadas
✅ Pantalla de inicio de sesión — S4

✅ Pantalla principal / Tablero de monitoreo — S5

⏳ Historial de registros + datos en tiempo real desde dispositivo portátil — S6

⏳ Interfaz adaptada para Android TV — S10-S12
